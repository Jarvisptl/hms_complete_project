# Hospital Management System (HMS) Capstone Project

A complete full-stack hospital management system built with Node.js, Express, MongoDB, and React/Vite.

## Project Overview

This system manages patient appointments, medical records, billing, and provides role-based access control for admin, doctor, and patient users.

## Technology Stack

**Backend:**
- Node.js
- Express.js
- MongoDB
- JWT Authentication

**Frontend:**
- React
- Vite
- CSS
